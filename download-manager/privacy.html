Privacy Policy

Effective date: June 14, 2026

In short: Download Manager does not use accounts, advertising SDKs, analytics SDKs, or developer servers. The app only connects to the direct file URL you provide in order to download the requested file.

1. Who We Are

Download Manager is developed by Halil Goktepe ("we", "us", "our"). This privacy policy explains how the Download Manager Android application handles information.

2. Data We Do Not Collect

We do not collect, sell, rent, or share personal data with the developer. Download Manager does not use:

  • Accounts or login
  • Advertising SDKs
  • Analytics SDKs
  • Crash reporting SDKs
  • Location access
  • Contacts access
  • SMS or call log access
  • Accessibility services

3. Data Stored On Your Device

The app stores download history locally on your device so you can see recent downloads and use actions such as open, share, retry, copy URL, or remove.

This local history may include the direct file URL you entered or shared with the app, the selected or inferred file name, and Android DownloadManager download IDs. This information is not sent to the developer.

4. Network Use

When you start a download, Android DownloadManager connects to the direct file URL you provided in order to download the requested file. The destination website or server may receive standard network request information, such as your IP address, user agent, and the requested URL, according to that website or server's own policies.

Download Manager does not proxy downloads through developer servers and does not upload downloaded files to the developer.

5. Security

Downloads from HTTPS links are transferred using the security provided by the destination website or server. If you choose to download from an HTTP link, that connection may not be encrypted.

6. Sharing Files

If you tap Share or Open, Android may pass the selected downloaded file to another app chosen by you. This only happens after a user action.

7. Third-Party SDKs

Download Manager does not include integrated advertising, analytics, attribution, or tracking SDKs.

8. Data Deletion

You can remove download history entries inside the app. You can also delete downloaded files from your device using Android system storage tools or by uninstalling the app.

9. Children's Privacy

Download Manager is not designed for children and is not directed to children under 13.

10. Changes to This Policy

We may update this privacy policy if the app's features change. Any changes will be reflected on this page with an updated effective date.
